HTML Basics

This section covers the basics of HTML.

HTML stands for HyperText Markup Language.

It is the standard language for creating web pages.

HTML elements are represented by tags.

Tags label pieces of content such as "heading", "paragraph", "table", and so on.

<h1>This is a heading</h1>
CSS Basics

This section covers the basics of CSS.

CSS stands for Cascading Style Sheets.

It describes how HTML elements are to be displayed on screen, paper, or in other media.

CSS saves a lot of work. It can control the layout of multiple web pages all at once.

External stylesheets are stored in CSS files.

body { background-color: lightblue; }
JavaScript Basics

This section covers the basics of JavaScript.

JavaScript is the programming language of the Web.

It can update and change both HTML and CSS.

JavaScript can calculate, manipulate and validate data.

It is an essential part of web development.

document.getElementById('demo').innerHTML = 'Hello JavaScript';
